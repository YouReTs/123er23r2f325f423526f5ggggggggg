1.  Переименуйте и скопируйтеь каталог autosurf-bootstrap-main в корень C:/autosurf-bootstrap/

2. 	Установите Node.js
 
3.  Откройте терминал(Power Shell) от администратора и введите команду для разрешения выполнения всех скриптов (не рекомендуется)
    Set-ExecutionPolicy Bypass   
 
4.	Установите Yarn командой в терминале:
    npm install -g yarn
	
5.	Установите Sandboxie Plus. Не выбирайте portable и включите автозапуск.

6.	Установите ProxyCap 5.35

7.	Скачайте portable версии нужных автосерферов.

8.	Поместите файлы автосерферов по пути autosurf-bootstrap/surfers/[surfer]/copy

9.	Запустите и настройте автосерферы. Эти настройки будут скопированы на каждую созданную копию программ. 
    Закройте все автосерферы перед следующим шагом.

10.	Настройте прокси для разных автосерферов в файлах autosurf-bootstrap/surfers/[surfer]/proxies.txt. 
    Для примера в этом файле есть два шаблона прокси.

11.	Настройте другие параметры в autosurf-bootstrap/config.ts. Для всех параметров есть комментарий с описанием.


12. Наконец откройте терминал в директории autosurf-bootstrap 
    cd C:/autosurf-bootstrap/
	
    и ввести команды 
	yarn install
	yarn run bootstrap
					 
(Если изменения настроек Proxycap не произошли ,разово восстановите файл настроек из директории C:\ProgramData\ProxyCap\machine.prs) и перезаргузите Windows

					 
Чтобы запустить все автосерферы, откройте сгенерированный start.bat. 
Он так же откроет вашу изначальную копию программы с вашего настоящего IP. Или выполните команду:
    yarn run start

Очистка
Для очистки системы от автосерферов, автозапуска и сброса настроек Sandboxie и ProxyCap выполните команду в директории autosurf-bootstrap:
    yarn run cleanup

Мониторинг
Вы можете включить мониторинг, который будет подробно показывать статистику по загрузке процессора и памяти на большом промежутке времени. Для этого выполните команду в директории autosurf-bootstrap:
    yarn run monitoring
